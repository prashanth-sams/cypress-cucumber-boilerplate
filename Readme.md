# Cypress | Cucumber Boilerplate

## Features
- [x] TS Linter on git commit
- [x] Page Object Pattern
- [x] Mocha HTML Report
- [x] Mocha XML Report
- [x] Allure Report
- [x] Screenshots on Failure
- [x] Hooks and Tags

### Installation
```
npm install
```

### Test Runner
```
npm run tags
```

### Launcher
```
npm run open
```
